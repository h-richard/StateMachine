This project implements the xDSL of a standard hierarchical UML state machine. 
It supports composite states and history states, yet no parallel states.
The metamodel has been edited with Xmodeling Studio allowing it to define business actions on states (entry, do, exit) and transitions.

This project comes with an example model of a functional microwave in StateMachine/model/microwave.stm.
Execution is launched using the StateMachineController.java in StateMachine/src
